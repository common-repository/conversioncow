Easily embed your ConversionCow campaigns in your WordPress website.

== Description ==

**[ConversionCow](https://www.conversioncow.com/) is a tool that adds a customizable slide-out page to your website, displaying social proof and call-to-actions.**

Use our no-code tools to quickly increase engagement and conversions on your website. Build popouts to quickly expose contact details, showcase unique statistics, or embed a form on your pre-existing website.

No coding experience required!

### Features

- **Brand-Friendly** - Match your existing website colors and imagery.
- **Conversion Templates** - Utilize our templates to get the most out of your popout.
- **UTM Targeting** - Show different popouts depending on UTM parameters.
- **A/B Testing** - Test different headlines, content and layouts.
- **Data Analytics** - Get specific reports for each campaign.

== Installation ==

1. Install the plugin using the WordPress plugin directory, or upload the plugin directly to your web server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'ConversionCow' settings page in WordPress.
4. Find your user code from the ConversionCow dashboard under the 'Install' page.
5. Copy and paste this code into the 'User Code' field in the ConversionCow settings page.

*Note: This plugin requires an active ConversionCow account.*

== Frequently Asked Questions ==

= Does this require an account? =

Yes, you'll need to create an account at [https://www.conversioncow.io/sign-up](https://www.conversioncow.io/sign-up) to use this plugin.

= Can I manage my campaigns from inside WordPress? =

No, you'll need to access the user portal at [https://conversioncow.io/](https://www.conversioncow.io/) to manage your campaigns.
